# AWS × Python × Vue ポートフォリオ

このリポジトリは、以下のスキルを確認していただくためのポートフォリオです。

- AWS Lambda を用いたサーバレス API 開発
- Python を用いたバッチ処理・自動化スクリプト
- Vue.js を用いた簡易ダッシュボード画面の実装

## プロジェクト一覧

### 1. lambda-hello-world

AWS Lambda（Python）で実装したシンプルな API のサンプルです。  
API Gateway から呼び出されることを想定し、JSON を返却します。

- 言語: Python 3.x
- 用途: サーバレス API の最小構成サンプル

### 2. python-csv-tool

CSV ファイルを読み込み、簡単な加工（合計値の計算など）を行うバッチスクリプトです。

- 言語: Python 3.x
- 用途: 業務でよくある CSV 加工・自動化スクリプトのイメージ

### 3. vue-dashboard-sample

Vue.js を用いた簡易ダッシュボード画面です。  
ダミーデータをグラフ風に表示し、「フロント側も触れます」ということを示すためのサンプルです。

- フレームワーク: Vue 3（CDN）
- 用途: 指標の一覧表示や簡単なダッシュボード UI のサンプル
